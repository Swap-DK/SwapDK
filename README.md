# SwapDK

Developer Kit for blockchain swap integrations

SwapDK is an open-source developer kit designed to simplify integration with
blockchain swap and liquidity infrastructure.
It provides a clean abstraction layer and utilities for building applications
that interact with cross-chain and on-chain swap systems.

## Features

- Unified interface for swap-related operations
- Modular and extensible architecture
- Designed for backend and SDK use
- Open-source and license-friendly
- Suitable for commercial and non-commercial projects
